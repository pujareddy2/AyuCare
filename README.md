# AyuCare

AyuCare is an Ayurvedic wellness web platform that blends traditional health guidance with modern, immersive frontend design.

## Live Demo

https://pujareddy2.github.io/AyuCare/

## Features

- AI plant scanner experience
- Ayurvedic recommendation module
- Food and diet tracker
- Meditation and yoga guidance
- Exercise and wellness suggestions
- Ayurvedic shop and cart flow

## Project Theme

Ancient Roots. Modern Wellness.
